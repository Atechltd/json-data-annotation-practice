# Annotation Exercise Guide

This folder contains structured practice exercises for AI data annotation and JSON validation.

## How to Use These Exercises

For each JSON file:

1. Identify incorrect fields
2. Validate all calculations (quantity × unit price)
3. Check subtotal, tax, and grand total consistency
4. Correct all dependent values
5. Ensure JSON structure remains valid

## Focus Areas
- Multi-field correction
- Invoice validation
- Mathematical accuracy
- Schema consistency
